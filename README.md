# Calorie Tracker

A web application to track daily calorie intake and exercise, built with React 19, TypeScript, and Tailwind.

## Features

- Register food and exercise activities
- Track calorie consumption and burn
- View net calorie summary
- Edit or delete activities
- Delete all activities with confirmation
- Dark/light theme toggle
- Persistent data via localStorage

## Tech Stack

- **React 19** with hooks (useReducer, useState, useEffect, useCallback)
- **TypeScript** strict mode
- **Tailwind CSS** with custom theme and dark mode
- **Vite 8** as build tool
- **uuid** for unique activity IDs
- **Heroicons** for icons
- **Google Fonts** (Bebas Neue + Outfit)

## Getting Started

```bash
npm install
npm run dev
```

## Build
